# Lab 2
In Lab 2, students shall use the IoT repository, information learned in lessons 1 and 2, and the command line to execute a series of commands as follows:

```
hostname
env
ps
pwd
git clone https://github.com/kevinwlu/iot.git
cd iot
ls
cd
df
mkdir demo
cd demo
nano file
cat file
cp file file1
mv file file2
rm file2
clear
man uname
uname -a
ifconfig
ping localhost
netstat
```
